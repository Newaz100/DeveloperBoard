Question-1: What are the different ways to select an element in the DOM?

You can select an element in the DOM using methods like getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), and querySelectorAll(). These methods allow selection by ID, class, tag, or CSS selectors.

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML gets or sets the HTML content inside an element, including tags. innerText retrieves or sets the visible text content, ignoring hidden elements. textContent also gets or sets the text content but does not account for styling like innerText.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where you attach a single event listener to a parent element, and it handles events for child elements. This improves performance and simplifies event management, especially for dynamically added elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism in which an event starts from the target element and bubbles up to its ancestors (parent elements) in the DOM tree. You can prevent this behavior using event.stopPropagation().

Question-5: How do you create, add, and remove elements using JavaScript?

You can create elements using document.createElement(), append them using appendChild() or insertBefore(), and remove them using removeChild() or remove(). These methods allow you to manipulate the DOM dynamically.
